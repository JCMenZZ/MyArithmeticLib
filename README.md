# MyArithmeticLib项目
### 项目介绍
该项目是针对日常学习工作中遇到的一些算法的总结和完善，
对这些算法进行总结后，在以后的运用中，我们就可以直接导入打好的`jar`包,
在需要用到的地方直接`import`即可使用。
### 注意事项
该包的目录结构请打开该文件进行查看，其中在每个子包下有一个文件夹**connector**，
该文件夹是存放不同算法的不同接口的，这里使用接口方便对不同地算法进行重写和覆盖。
这里不会将单元测试文件夹删除，方便具体查看方法的使用。如不需要请忽略。

+ **sort**下存放的是排序的算法，例如：（冒泡排序算法等）
